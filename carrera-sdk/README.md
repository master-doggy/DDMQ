## DDMQ SDK ##
DDMQ provides producer&consumer SDK in major programming languages. Since most of the logics and features are implemented in proxy side, SDK is relatively easy to implement.

producer SDK：

* Java
* Go
* PhP
* C/C++ 
* Python

consumer SDK:

* Java
* Go
* C/C++
